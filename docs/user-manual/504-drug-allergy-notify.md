# 504 - บันทึกแพ้ยา-ยาเฝ้าระวัง

2. จะปรากฎข้อมูลแพ้ยา-ยาเฝ้าระวัง ของผู้ป่วย

2. การเพิ่มหรือแก้ไขข้อมูลการแพ้ยา-ยาเฝ้าระวัง
   กดปุ่ม ![Logo](./img/image504-3.png) 
   จะปรากฎหน้า เพิ่ม/แก้ไข > กดปุ่ม "บันทึก"
